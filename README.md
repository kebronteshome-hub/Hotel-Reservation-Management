# Hotel-Reservation-Management
## Our Project Overview

The Hotel Reservation Management System is a simple Java console application developed using Object-Oriented Programming (OOP) principles. The system helps manage hotel rooms, customers, employees, and reservations through a menu-driven program.

We created this project to practice Java programming concepts such as classes, inheritance, encapsulation, constructors, and ArrayLists.

---

## Main Features

* Display available hotel rooms
* Make room reservations
* View reservation records
* Store customer information
* Store employee information
* Simple user interaction using the console

---

## OOP Concepts Used From The Lessons We Learned:

### Classes and Objects(From V1.0 & V2.0

The project is built using multiple classes such as `Hotel`, `Room`, `Customer`, and `Reservation`. Objects are created from these classes to perform different tasks.

### Inheritance(From V1.0 & V2.0)

`Customer` and `Employee` classes inherit from the `Person` class. This helps reduce code duplication and improves reusability.

### Encapsulation(From V3.0)

Class attributes are declared private and accessed using public getter methods to protect data.

### Method Overriding(From V4.0)

The `displayInfo()` method is overridden in both `Customer` and `Employee` classes to display specific information.

### ArrayList

The program uses `ArrayList` to store rooms and reservations dynamically.
